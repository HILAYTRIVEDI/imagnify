<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repo</p>

```
git clone https://github.com/HILAYTRIVEDI/imagnify.git
```

<p>2. Install depedencies</p>

```
npm install
```

<p>3. Run the app</p>

```
npm run dev
```
